## Responsive Strategy

<details markdown="1">
<summary>Click to see details</summary>

- Uses mobile-first Tailwind utility classes.
- Padding adapts based on screen size (`p-6` for mobile, `p-8` for tablets/desktops).
- Flexbox for column-based structural layout.

</details>
