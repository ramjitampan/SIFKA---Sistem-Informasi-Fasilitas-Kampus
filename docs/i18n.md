# Internationalization (i18n)

## Overview
The application uses `i18next` for internationalization.

## Configuration
- Located in `FE_SIFKA/src/i18n.js`.
- Supported Languages: English (`en`), Indonesian (`id`).
- Fallback: English (`en`).

## Adding New Translations
1. Create/update files in `FE_SIFKA/src/locales/{lang}/`.
2. Import the new file in `FE_SIFKA/src/i18n.js` and add it to the `resources` object.

## Usage in Components
```jsx
import { useTranslation } from 'react-i18next';

const Component = () => {
  const { t } = useTranslation('common');
  return <h1>{t('welcome')}</h1>;
};
```
